## Projeto desenvolvido como parte do desafio #7DaysOfCode da Alura. O objetivo foi criar um site para a empresa fictícia OptimusTech, em 7 dias, seguindo um layout fornecido no Figma.
![tumblr_1f70dd4d9d6ead691c24da3b9bec99ee_4e09d93e_540](https://github.com/user-attachments/assets/70a31ffb-b127-45a9-b7c8-cdf3fd16b08b)

## Como esse desafio funcionou? 
### Durante uma semana, uma vez por dia, eu recebi o desafio de criar uma seção de cada vez do site para a OptimusTech (header, seção de vagas, rodapé, etc...) seguindo o objetivo principal de entregar um site para a procura de novos talentos na empresa.
![tumblr_ad8728aec1b1fe26cb395f9411c30776_984bfee5_540](https://github.com/user-attachments/assets/7b345cb0-7a7d-44b3-8112-23e9614f6eb5)

## Tecnologias Utilizadas:
### Este projeto foi desenvolvido utilizando as seguintes tecnologias web:
### HTML: O esqueleto da estrutura, garantindo o funcionamento de cada parte do projeto. 
### CSS: A cara do projeto, utilizando o design escolhido pela empresa para passar ao futuro colaborador a alma da equipe.
